À propos de cette application

Cette application Flask minimaliste a été créée pour démontrer les bases d'une architecture propre.

Principales caractéristiques :

  • Utilisation de Flask pour le backend
  • Configuration via variables d'environnement (.env)

  • CSS simple pour un design responsive

Cette application peut être utilisée comme point de départ pour vos propres projets.
